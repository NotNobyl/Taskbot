# Building the TaskBot Android APK

## What you need to install first (one-time)

- **Node.js** — nodejs.org (LTS version)
- **Android Studio** — developer.android.com/studio (~1GB)
  - During install, accept the Android SDK and emulator components

---

## Step 1 — Update your Railway URL in index.html

Open `static/index.html` and find this line near the top of the script:

```js
const API = 'https://taskbot-production.up.railway.app';
```

Replace with your actual Railway URL.

---

## Step 2 — Set up Firebase (for push notifications)

1. Go to **console.firebase.google.com** → Add project → name it `taskbot` → Continue
2. Skip Google Analytics → Create project
3. Click the **Android icon** (</>) to add an Android app
   - Package name: `com.jacob.taskbot`
   - App nickname: TaskBot
   - Click **Register app**
4. Download `google-services.json` — save it in your `android/app/` folder (you'll create this in Step 4)
5. Skip the rest of the Firebase wizard for now

**Get your service account key (for the backend):**
1. In Firebase → Project Settings (gear icon) → Service accounts
2. Click **Generate new private key** → Download the JSON file
3. Rename it `firebase-service-account.json` and put it in your taskbot folder
4. Add it to Railway: Variables → `FIREBASE_SERVICE_ACCOUNT` → paste the full JSON content as the value

---

## Step 3 — Install Capacitor

In your taskbot folder, open a terminal:

```bash
npm install
npx cap init TaskBot com.jacob.taskbot --web-dir static
```

---

## Step 4 — Add the Android platform

```bash
npx cap add android
```

This creates an `android/` folder — that's your Android Studio project.

Now copy the `google-services.json` you downloaded from Firebase into:
```
android/app/google-services.json
```

---

## Step 5 — Open in Android Studio

```bash
npx cap open android
```

Android Studio opens. Wait for it to finish syncing (progress bar at the bottom — takes a minute or two).

---

## Step 6 — Add Firebase to the Android project

In Android Studio, open `android/app/build.gradle` and add inside `dependencies {}`:

```gradle
implementation platform('com.google.firebase:firebase-bom:32.7.0')
implementation 'com.google.firebase:firebase-messaging'
```

Then open `android/build.gradle` and add to `dependencies {}` under `buildscript`:

```gradle
classpath 'com.google.gms:google-services:4.4.0'
```

And add to the bottom of `android/app/build.gradle`:

```gradle
apply plugin: 'com.google.gms.google-services'
```

Click **Sync Now** when Android Studio prompts you.

---

## Step 7 — Build the APK

In Android Studio:
1. Menu → **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
2. Wait for the build (1-2 minutes)
3. Click **locate** in the popup — it opens the folder with your `.apk` file

The APK is at: `android/app/build/outputs/apk/debug/app-debug.apk`

---

## Step 8 — Install on your phone

**Option A — USB:**
1. On your Android phone: Settings → Developer Options → enable USB Debugging
   (To enable Developer Options: Settings → About Phone → tap Build Number 7 times)
2. Plug in via USB
3. In Android Studio: **Run** → **Run 'app'** → select your device

**Option B — Copy the APK file:**
1. Copy `app-debug.apk` to your phone (via USB, Google Drive, email, etc.)
2. On your phone: open the file → tap Install
3. If blocked: Settings → Security → enable **Install unknown apps** for your file manager

---

## Step 9 — Enable push notifications in the app

1. Open TaskBot on your phone
2. Tap the 🔔 bell button
3. Allow notifications when prompted
4. Done — reminders will now push to your phone even when the app is closed

---

## Updating the app later

When you change code:
```bash
# After editing static/index.html or other web files:
npx cap sync android

# Then rebuild the APK in Android Studio (Step 7)
```

## File structure after setup

```
taskbot/
├── app.py
├── discord_bot.py
├── database.py
├── requirements.txt
├── Procfile
├── package.json
├── capacitor.config.json
├── firebase-service-account.json   ← keep this private, don't commit to GitHub
├── static/
│   ├── index.html
│   ├── sw.js
│   └── manifest.json
└── android/                        ← generated by Capacitor
    └── app/
        ├── google-services.json    ← from Firebase
        └── build/outputs/apk/...  ← your APK lives here
```

> ⚠️ Add `firebase-service-account.json` to your `.gitignore` — it's a secret key.
